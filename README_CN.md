## 📖 关于 ProxyCap-CN

ProxyCap-CN 是 [ProxyCap](https://www.proxycap.com/index.html) 软件的中文本地化版本，旨在帮助中国用户更便捷地使用 ProxyCap 的强大功能，提升用户体验。

---

## 📋 汉化文件使用方法

汉化文件使用非常简单，只需以下两步即可完成：
1. 复制 `pcapui.exe` 文件
2. 粘贴到安装目录（例如：`D:\Program Files\Proxy Labs\ProxyCap`）覆盖同名文件
3. 启动软件，即可享受中文界面
