## 📖 项目介绍

ProxyCap-CN 是 [ProxyCap](https://www.proxycap.com/index.html) 的中文本地化版本，旨在帮助中国用户更便捷地使用 ProxyCap 的强大功能，提升用户使用体验。

## 🚀 快速开始

### 📥 安装步骤

1. **下载安装** - 从官方网站下载并安装 ProxyCap
   - 📎 [官方下载页](https://www.proxycap.com/download.html)

2. **替换文件** - 将汉化文件复制到安装目录
   ```bash
   # 复制 pcapui.exe 文件
   ProxyCap-version-5.42-x64/zh/pcapui.exe
   # 粘贴到安装目录覆盖原文件
   D:\Program Files\Proxy Labs\ProxyCap\pcapui.exe
   ```

3. **完成汉化** - 启动 ProxyCap 即可享受中文界面

## ✨ 主要特性

| 特性 | 描述 |
|------|------|
| 🔌 **代理协议** | 支持 SOCKS、Shadowsocks 和 HTTPS 代理服务器 |
| 🌐 **协议支持** | 支持 TCP 和 UDP 网络协议 |
| 🔒 **SSH 隧道** | 内置 SSH 隧道支持 |
| 🔍 **DNS 解析** | 代理端 DNS 解析 |
| 📊 **路由规则** | 灵活的路由规则配置 |
| ⛓️ **代理链** | 支持多级代理链 |
| 💻 **64位支持** | 完美支持 64 位应用程序 |
| 📡 **IPv6** | 支持 IPv6 协议 |
| 🔐 **SSO** | 支持单点登录 |
| ✅ **代理检测** | 内置代理检测工具 |
| 🏢 **集中管理** | 支持集中化管理 |

## 📚 软件介绍

ProxyCap 可以让你将计算机的网络连接通过代理服务器进行转发。你可以让 ProxyCap 控制特定的应用程序在特定情况下通过代理连接互联网。

✨ **无需重新配置任何互联网客户端**，这一切都通过友好的用户界面完成。ProxyCap 原生支持 SSH 协议，允许你指定 SSH 服务器作为代理服务器。
