## 关于 ProxyCap-CN

ProxyCap-CN 是 ProxyCap 的中文本地化版本，旨在帮助中国用户更便捷地使用 ProxyCap 的强大功能，提升用户使用体验。

**本项目仅为本地化翻译项目，不包含任何破解、补丁或任何形式的激活手段，所有权利归 Proxy Labs 所有。**

## 安装说明

1. 从官方网站下载并安装 ProxyCap
2. 将 `ProxyCap-version-5.42-x64/zh/` 目录下的 `pcapui.exe` 文件复制出来
3. 粘贴到 ProxyCap 安装目录下（如 `D:\Program Files\Proxy Labs\ProxyCap`）
4. 替换原有的 `pcapui.exe` 文件
5. 启动 ProxyCap 即可享受中文界面

## 软件介绍

ProxyCap 可以让你将计算机的网络连接通过代理服务器进行转发。你可以让 ProxyCap 控制特定的应用程序在特定情况下通过代理连接互联网。这一切都通过友好的用户界面完成，无需重新配置任何互联网客户端。ProxyCap 原生支持 SSH 协议，允许你指定 SSH 服务器作为代理服务器。

### 主要特性

- 支持 SOCKS、Shadowsocks 和 HTTPS 代理服务器
- 内置 SSH 隧道支持
- 支持「纯」HTTP 代理
- 支持 TCP 和 UDP 网络协议
- 代理端 DNS 支持
- 灵活的路由规则
- 支持代理链
- 支持 64 位应用程序
- IPv6 支持
- 支持单点登录（SSO）
- 内置代理检测工具
- 集中管理支持
