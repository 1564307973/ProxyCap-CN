## About ProxyCap-CN

ProxyCap-CN is the Chinese localization version of [ProxyCap](https://www.proxycap.com/index.html) software, designed to help users in China more easily access ProxyCap's powerful features and enhance their user experience.

> ⚠️ **Disclaimer**: This project is purely for localization purposes only. It does not include any cracks, patches, or any form of software activation. All rights belong to [Proxy Labs](https://www.proxycap.com/index.html).

---

## Usage Instructions

To use the localized file, copy the `pcapui.exe` file to the installation directory, such as "D:\Program Files\Proxy Labs\ProxyCap", and overwrite the existing file to complete the localization.
